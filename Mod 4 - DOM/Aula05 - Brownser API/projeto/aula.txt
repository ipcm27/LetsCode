Ao final da aula você aprenderá


    * Como realizar uma consulta GET com fetchApi
    * Como preencher uma tabela com dados da Api
    * Como verificar dados de API's com o Postman
    * O que é e para que serve e como utilizar o Async e Await
    * O que é uma promise e como utilizar 
    * Consumir uma promise já existente.
    * Utilizar o promise all
    * Operações CRUD com LocalStorage Browse, Read, Edit, Add and Delete
    * Diferença entre Session Storage, Cookies e LocalStorage
